# SearchFromList
Search Strings From Array and Display The String , Position in array , And The SizeOf Array

Example:

# ENTER NAME OF CAR TO SEARCH:
    Benz
     (Benz Maybach, 0 , 9 )


         DETAILS
======================
# NAME OF CAR : Benz Maybach
# POSITION IN ARRAY : 0
# SIZE OF ARRAY : 9
